# 📸 팀 사진 가이드

각 구단의 **누끼가 제거된 PNG 이미지**를 이 폴더에 추가해주세요!

## 📁 필요한 파일 목록:

- `doosan.png` - 두산 베어스
- `kia.png` - KIA 타이거즈
- `hanwha.png` - 한화 이글스
- `samsung.png` - 삼성 라이온즈
- `lg.png` - LG 트윈스
- `kiwoom.png` - 키움 히어로즈
- `lotte.png` - 롯데 자이언츠
- `ssg.png` - SSG 랜더스
- `nc.png` - NC 다이노스
- `kt.png` - KT 위즈

## 🎨 이미지 요구사항:

1. **배경 투명 (PNG)** - 누끼 제거 필수
2. **크기**: 최소 800x800px (정사각형 권장)
3. **용량**: 300KB 이하 (최적화)
4. **내용**: 팀 로고 or 마스코트 or 대표 선수

## 💡 추천 이미지:

- 팀 공식 로고 (누끼 제거)
- 마스코트 캐릭터
- 팀 단체 사진 (누끼 제거)
- 대표 유니폼

## 🔧 누끼 제거 도구:

- https://www.remove.bg/ (자동 누끼 제거)
- Photoshop
- GIMP (무료)
